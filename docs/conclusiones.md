# Conclusiones

Hacer esta actividad me ha servido para practicar Markdown, Git y GitHub Actions de manera real. Documentar todo el proceso me ayudó a organizar la información y ver cómo se estructura un proyecto.  

Trabajar con ramas y GitHub Pages me enseñó a mantener los cambios controlados y a publicar documentación fácilmente. Además, montar un contenedor NGINX con Docker, aunque no sea necesario para la página, me permitió entender cómo funcionan los contenedores, los puertos y los volúmenes.  

En resumen, esta tarea ha sido útil para ver cómo se conectan varias herramientas y conceptos de SecDevOps de forma práctica.
